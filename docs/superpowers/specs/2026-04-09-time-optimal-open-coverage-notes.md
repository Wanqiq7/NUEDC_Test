# 时间最优开放巡查与斜降落约束说明

- 题面“巡查完成时间不得超过 300s，越快越好”已经映射为任务级飞行代价最小化，不再将“闭环”或“最少重复格”视为首要目标。
- 题面“完成巡查后，无人机以 45°±5° 俯角准确稳定降落到起飞区域”已经映射为终点候选区域约束，终点必须处于可接入斜降落走廊的方格集合内。
- 图 1 中红色起飞区位于右下侧，因此样例配置不再错误地使用 `A9B7` 作为固定起终点，而改为右下侧入场起飞语义。
- 当前实现采用“开放巡查路径 + 终点接驳代价 + 斜降落兼容性惩罚”的组合目标函数，用于比较候选终点。
- 当前实现仍保留 legacy 规划器作为回退路径，以便在题意建模和 UI 联调阶段维持系统稳定。
